# kas.kaplingan

A simple **web-based cash management system** for tracking installment or community fund payments (kas kaplingan).  
This application helps record member payments, monitor total funds, and manage simple financial tracking for small communities or groups.

🌐 **Live Demo**  
https://kas-kaplingan.vercel.app/

---

# 📌 Features

- 💰 Record member payments
- 📊 Automatic total cash calculation
- 📋 Display payment records in a list
- 🗑 Delete or manage payment entries
- 📱 Responsive design (desktop & mobile friendly)

---

# 🛠 Tech Stack

This project is built using simple web technologies:

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **Vercel** for deployment
